# expense-tracker-python
a command-line expense tracker built with python that tracks expenses.
## What it does
- Accepts unlimited expenses entries
- shows all the expenses enteries made
- Calculates total amount spent 
- stops when you choose option "4. Quit"
## What I learned building this
- Used a while loop to keep the menu running until the user chooses to quit
- Used append to store each expense into a list
- Used sum to calculate the total amount spent automatically
- Used dictionaries to store category, amount and description together
- Used float() to accept decimal amounts
## How to run it
1. Make sure Python is installed
2. Download 'result_calculator.py'
3. Open terminal and run:
python expense_tracker.py
